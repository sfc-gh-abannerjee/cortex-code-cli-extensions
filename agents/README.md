---

## Agent vs. Skill

**Use an Agent when:**
- Task requires custom tools
- Need to maintain state across steps
- Complex multi-step workflows
- Research or exploration tasks

**Use a Skill when:**
- Simple to moderate workflows
- Can use Cortex Code's built-in tools
- Stateless operations
- Pre-commit checks, validation, formatting

See the [Cortex Code documentation](https://docs.snowflake.com/user-guide/snowflake-cortex/cortex-code) for more details.
